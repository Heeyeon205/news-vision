## Overview
- 뉴션은 읽기 어려운 뉴스를 뉴션 공식 계정과 크리에이터들이 쉽게 풀어서 보여주는 플랫폼입니다.
- 바쁜 현대인들을 위한 탑 10 뉴스를 3줄 요약해서 확인 할 수 있습니다.
- 커뮤니티에서 뉴스에 대해 지적인 소통과 경험을 공유할 수 있습니다.
- 뉴스 정보 혹은 사회 현상에 대한 투표를 할 수 있는 콘텐츠를 제공합니다.
